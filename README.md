# Devon Purification - Portfolio

Personal portfolio website showcasing my work as a Software Developer & QA Engineer.

## Tech Stack

- **React 18** - UI framework
- **Vite** - Build tool and dev server
- **CSS** - Custom styling with design tokens
- **Vercel** - Hosting

## Features

- Single-page layout with smooth scroll navigation
- Fixed sidebar with active section tracking
- Responsive design (mobile-friendly)
- Page load animations
- Futuristic HUD-style UI design

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This site is deployed on Vercel. Push to `main` to trigger automatic deployment.
